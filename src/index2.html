SMART HOME IOT BLOCKCHAIN

ABOUT

Smart Home IoT is a blockchain based application which aims at securing the data coming from sensors used in a smart home. Sensors such as DHT-11, HC SR04 etc have been used to track various parameters which are necessary to ensure safety at home. Blockchain technology is used to store the data from these sensors, hence making the data completely secure and tamper-proof. This will ensure that hackers won't be able to hack into the system and cause issues.
